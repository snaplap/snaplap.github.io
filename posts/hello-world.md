# Hello World

欢迎来到我的个人博客！这是写下的第一篇文章。

## 为什么要写博客？

写博客有很多好处：

1. **记录知识** — 将学到的东西整理下来，加深理解
2. **分享经验** — 与他人分享自己的经验和见解
3. **建立习惯** — 养成定期总结和思考的习惯

## 我打算写什么？

主要包括以下几个方向：

- 技术学习笔记（前端、后端等）
- 项目实战经验分享
- 读书笔记与感悟
- 生活随笔

## 如何给博客添加新文章？

只需要两步：

1. 在 `posts/` 目录下新建一个 `.md` 文件，写好 frontmatter
2. `git add . && git commit -m "新文章" && git push`

GitHub Action 会自动更新文章索引，几秒钟后博客就更新了。

> 好记性不如烂笔头。让我们开始吧！🚀
